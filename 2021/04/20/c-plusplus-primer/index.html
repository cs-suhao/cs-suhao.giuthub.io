2.1 基本内置类型

2.1.1 算术类型

  • bool 布尔类型
  • char 字符 8bit
  • wchar_t 宽字符 16bit
  • char16_t Unicode字符 16bit
  • char32_t Unicode字符 32bit
  • short 短整型 16bit
  • int 整型 16bit
  • long 长整型 32bit
  • long long 长整型 64bit
  • float 单精度浮点数 6位有效数字
  • double 双精度浮点数 10位有效数字
  • long double 扩展精度浮点数 10为有效数字

类型选择的建议：

  1. 明确数值不为负值，用无符号类型。
  2. int不够用，选择long long。
  3. 使用unsigned char和signed char。
  4. 浮点数运算首选double。

2.1.2 类型转换

  • 如果给无符号类型赋值，超出了它的表示范围，则初始值是对无符号类型表述数值总数取模后的余数。
  • 无符号类型和带符号类型不要混用，因为会出现类型转换的结果。

2.1.3 字面值常量

整型和浮点类型字面值常量分为十进制、八进制和十六进制，其中以0开头的表示八进制，0x或0X开头的为十六进制。例如同样表示20，用不同的进制如下：


默认情况下，十进制字面值是带符号数，八进制和十六进制既可能是带符号也可能是无符号。

字符和字符串字面值区分标志为单引号和双引号，并且字符串字面值的类型实际上是由常量字符构成的数组，编译器会在每个字符串的结尾添加一个空字符'\0'。
